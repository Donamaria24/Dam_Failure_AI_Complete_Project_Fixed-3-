# AI-Powered Dam Failure Early Warning System

This project includes:
- Crack detection using YOLOv8 and phone camera
- Model training on crack segmentation dataset
- Integration with IP Webcam app for real-time detection

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Train the model:
   python train_model.py
3. Start IP Webcam app on your phone
4. Update phone IP in crack_detection.py
5. Run detection:
   python crack_detection.py
